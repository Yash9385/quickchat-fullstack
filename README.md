# QuickChat – Full Stack Chat Application

QuickChat ek full-stack real-time chat application hai.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Realtime: Socket.io

## Project Structure
- client/  → Frontend
- server/  → Backend

## How to Run Project

### Backend
cd server
npm install
npm start

### Frontend
cd client
npm install
npm run dev

## Environment Variables
Create `.env` files in client and server folders.
Do NOT push `.env` to GitHub.
